  Zelda-Style 2D World Builder

  A browser-based editor for creating top-down 2D games like Zelda, Pokemon, or Stardew Valley.

  Getting Started

  1. Download or clone this repo
  2. Open world-builder.html in your browser
  3. That's it! No install needed.

  How to Use

  Loading Assets

  - Click "Upload Tileset" to add your tile images
  - Get free assets from https://itch.io/game-assets/tag-tileset or ask Khabe for some!

  Building Your Map

  1. Tiles Tab - Select tiles from your tileset and paint on the map
  2. Layers - Add multiple layers (ground, objects, overhead)
  3. Collision - Click "Edit Tile Collisions" to mark solid tiles

  Adding NPCs

  1. Go to NPCs Tab
  2. Click "+ Create NPC" and upload a sprite sheet
  3. Select frames for each animation (Walk Down, Up, Left, Right, Idle)
  4. Place NPCs on the map
  5. Draw paths for them to walk
  6. Set triggers: Loop, On Interact, Day Only, or Night Only
  7. Click "Preview" to watch them walk in the editor!

  Lighting

  - Go to Lights Tab
  - Set time of day (Dawn, Day, Dusk, Night)
  - Place point lights (torches, lanterns)
  - Enable day/night cycle

  Sounds

  - Go to Sounds Tab
  - Upload audio files
  - Place ambient sounds on the map
  - Set player walk/attack sounds

  Testing

  - Click "Test Map" to play your game
  - Arrow keys to move, Space to attack
  - Press P for debug settings

  Saving/Loading

  - Save - Downloads your project as a JSON file
  - Load File - Load a previously saved project
  - Your player sprite and all assets are embedded in the save!
